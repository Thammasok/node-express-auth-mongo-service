# node-express-auth-mongo-service

A simple authentication service using Node.js, Express, MongoDB, and JWT.

## Setup

install dependencies

```bash
pnpm install
```

run the service

```bash
pnpm dev
```

server will be running on port 3000

## Services

- [x] Login
- [x] Signup
- [x] Account
- [x] Verify Email
- [x] Forgot Password
- [x] Reset Password
- [x] Logout
- [x] Refresh Token
- [x] Refresh Token Expiration
